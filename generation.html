Instructions
  1. Go to ADO and make a query that gets the work items you want to generate a planning poker game for. Copy the query's GUID out of the URL in ADO.
  2. Go to Thunderdome, make an account if you don't have one, and generate an api key by clicking on your username and then the API key button.
  3. Paste those two things into the above text boxes
  4. Enter your team name into the third text box
  5. Bookmark the page that you get when you submit the form to quickly generate another planning poker game for the same query
